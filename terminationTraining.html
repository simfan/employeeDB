Termination Training Checklist
  1. SAFETY
    • All guards must be in place when machine is turned on
    • Keep hands out of the guard area when the machine is turned on
    • Keep the floor around the press clear of bins, drums, racks, etc.
    • Be sure the machine is turned off when leaving the area or when not working
    • Never stick finger into die, use pick or other tool to unjam or clear tooling
    • Machine must be unplugged when changing dies or tooling
  2. PAPERWORK
    • Complete daily Production Papers with proper codes, part number, lot number, and times
  3. TERMS
    • crimp height
    • crimp width
    • core crimp
    • insulation crimp
    • insulation height
    • insulation width
    • window
    • first piece inspection
    • tooling
    • brush
    • terminals - TB, TQ, TS, TP, TF, TX, TR, TU, TM
    • back feed die
    • side feed die
    • air feed die
    • pull test
    • micrometer
    • caliper
    • serial tracking
  4. CUTTING CARD AND CRIMP VERIFICATION
    • Compare cutting card information against wire to be terminated - make sure wire, gauge, color, strip length, hot stamp, or other wire markings are correct
    • Obtain correct terminals and terminate test wire unless otherwise instructed - long strip must be used so no insulation will influence the pull test results
    • Measure crimp height - adjust the crimp height if necessary or have set-up person or supervisor adjust the press to meet measurements on cutting card
    • When pull testing a terminated wire, the fingers that hold the terminal in the pull test should not come in contact with the terminal crimps
    • Write crimp height and pull test results on cutting card and fill out the set-up log
  5. QUALITY
    • Explain Fargo Quality Procedure(FP 122B) within 2 weeks
    • Watch for twisting or bending of terminal
    • No strands should be outside of crimp area
    • Watch for smashed or crushed areas on terminal - examine the mating area of the terminal to make sure it is not damaged - be sure terminal is cut off reel properly
    • Watch for insulation in core crimp area
    • All crimsps should have acceptable window and brush
    • If specified, insulation crimp must be measured and meet specifications on cutting card
    • Watch for distorted locking tab
    • Use tooking block to check for terminal clearance if one is provided
  6. REPAIRS
    • If a mistake is made while terminating, cut the terminal off between the insulation and core crimps
    • Restrip the wire to the proper length and make sure wire length is within tolerance after re-stripping
    • Be sure new strip cutoff is a straight, no-angle cut
    • Be sure there are no cut or nicked strands after re-stripping
    • Complete scrap paper if an unusual amount of wire or terminals is discarded in the setup and/or production process
  7. CLEAN UP
    • Clean up the machine at the end of job or shift
    • Return all tools to proper place
    • Sweep floor, return empty bins to proper area, log out any empty reels
    • Return any tooling or die parts to proper places or give to set-up person or supervisor
  8. HOW TO TRAIN
    • Trainer explains
    • Trainer explains and shows
    • Trainer performs along with employee
    • Trainer observes employee
    • Employee performs on own, trainer checks on progress
    • Trainer is always available for questions
